# Skill Market — 技能市场平台

一个用于分享、发现和管理 AI Agent 技能的在线平台，支持技能的创建、上传、搜索、评分与下载。

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React + TypeScript + Vite |
| 后端 | Python + Flask |
| 数据存储 | JSON 文件存储 |

## 项目结构

```
├── frontend/          # 前端项目（React + Vite）
├── backend/           # 后端项目（Flask API）
├── docs/spec/         # 产品规格文档
└── .spec-workflow/    # Spec 工作流模板
```

## 快速开始

### 后端

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### 前端

```bash
cd frontend
npm install
npm run dev
```
