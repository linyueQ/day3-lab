## 1. 性能需求

### 1.1 延迟 SLO

| 指标 | 目标 | 度量 |
|------|------|------|
| `GET /skills` P95 | < **300ms** | 服务端 |
| `GET /skills/<id>` P95 | < **200ms** | 服务端 |
| `POST /skills` P95 | < **500ms** | 服务端 |
| `POST /skills/upload`（10MB zip）P95 | < **30 秒**（含解压） | 服务端 |
| `POST /skills/draft`（LLM 草稿）P95 | < **10 秒** | 服务端 |
| `POST /skills/smart-search` P95 | < **10 秒** | 服务端 |
| 互动类 `POST /like/favorite/rate` P95 | < **300ms** | 服务端 |
| 列表首屏 LCP（本地） | < **2.5s** | Chrome DevTools |
| 主题切换到全站响应 | < **100ms**（纯 CSS 变量） | 用户感知 |

### 1.2 LLM 与降级

| 场景 | 降级 |
|------|------|
| `POST /skills/draft` LLM 超时/异常 | 返回 `fallback=true` + 本地模板草稿；HTTP 200 |
| `POST /skills/smart-search` LLM 超时/异常 | 自动调用内部关键词搜索补足到 ≥5 条；`fallback=true`；HTTP 200 |
| LLM 连续失败 N 次 | 断路器 60 秒内直接降级 |

### 1.3 热度重算

| 项 | 规格 |
|----|------|
| 重算触发 | 互动写入后同步更新 `hot_score` 字段 |
| 重算策略 | 写时更新（write-through）；不单独后台任务 |
| 一致性 | 互动 API 返回 200 后，1 秒内下次 `sort=hot` 反映新顺序 |

## 3. 可观测性

### 3.1 traceId

| 项 | 规格 |
|----|------|
| 格式 | `tr_{uuid4.hex}` |
| 注入 | 所有成功响应顶层 + 错误 `error.traceId` |
| LLM 调用 | 以子 span 形式日志 `trace_id / upstream_latency_ms / fallback_used` |

### 3.2 错误码

| HTTP | `error.code` | 触发 |
|------|--------------|------|
| 400 | `EMPTY_FIELD` | 必填缺失 |
| 400 | `INVALID_CATEGORY` | category 非枚举 |
| 400 | `INVALID_TAG` | tags 超限 |
| 400 | `INVALID_QUERY` | q/intent/smart query 超长 |
| 400 | `INVALID_RATING` | 评分非 1-5 |
| 400 | `MISSING_SKILL_MD` | zip 根目录无 skill.md |
| 400 | `BUNDLE_LIMIT_EXCEEDED` | 解压超限 |
| 400 | `UNSAFE_ZIP` | zip 含 `..`/绝对路径 |
| 400 | `INVALID_PAGE` | page_size 超限 |
| 404 | `SKILL_NOT_FOUND` | id 不存在 |
| 413 | `FILE_TOO_LARGE` | zip > 10MB |
| 429 | `RATE_LIMITED` | 限速触发 |
| 500 | `UPSTREAM_ERROR` | 未预期异常 |

### 3.3 日志

| 事件 | 字段 |
|------|------|
| 请求入/出 | method / path / status / duration_ms / ip / visitor_id / traceId |
| 上传审计 | skill_id / size / file_count / traceId |
| LLM 调用 | endpoint / upstream_status / upstream_latency_ms / fallback_used |
| 互动 | skill_id / visitor_id / action (like/favorite/rate) / new_value |

## 4. 安全需求（摘要，详见 `11`）

| 项 | 当前 |
|----|------|
| API 认证 | 无（v0.2 匿名，`visitor_id` cookie 身份） |
| CORS | 全开放（教学） |
| ZIP 解压 | zip-slip 防护 + 大小/数量硬上限 |
| XSS | 双层消毒（marked + bleach + DOMPurify） |
| LLM 安全 | 输入转义、系统 Prompt 隔离、输出字段白名单 |

### 4.1 输入校验清单

| 字段 | 规则 |
|------|------|
| name | 1–80 字符 |
| category | 枚举：`frontend / backend / security / bigdata / coding / ppt / writing / other` |
| tags[] | 0–6，每项 1–20 字符，`[a-z0-9\-\u4e00-\u9fa5]+` |
| description | 1–200 字符 |
| skill_md | 1–10000 字符 |
| intent（LLM 输入） | 10–200 字符 |
| q / smart query | 1–200 字符 |
| page_size | 1–50 |
| rating | 1–5 整数 |
| zip size | ≤ 10MB |
| zip 解压总大小 | ≤ 50MB |
| zip 文件数 | ≤ 200 |
| 解压目录 | 限定 `./data/bundles/<skill_id>/` |

### 4.2 限速

| 端点 | 限制 |
|------|------|
| `POST /skills/upload` | 每 visitor 每分钟 ≤ 3 次 |
| `POST /skills` | 每 visitor 每分钟 ≤ 5 次 |
| `POST /skills/draft` | 每 visitor 每 5 秒 ≤ 1 次 |
| `POST /skills/smart-search` | 每 visitor 每 3 秒 ≤ 1 次 |
| `POST/DELETE /skills/<id>/like\|favorite` | 每 visitor 每秒 ≤ 5 次 |

## 5. 数据约束

### 5.1 类型约束

| 字段 | 类型 | 约束 |
|------|------|------|
| `view_count / download_count / like_count / favorite_count / rating_count` | integer | ≥ 0 单调递增 |
| `rating_avg` | float | 0.0–5.0，保留 2 位小数 |
| `hot_score` | float | ≥ 0 |
| `category` | string | 枚举 |
| `tags` | array<string> | 去重 |
| `created_at / updated_at` | string | ISO-8601 UTC |
| `theme`（前端） | string | `light` / `dark` |

### 5.2 排序公式

| sort | 表达式 |
|------|--------|
| `hot` | `0.5 × norm(download_count) + 0.3 × norm(rating_avg × min(rating_count,50)/50) + 0.2 × norm(like_count + favorite_count)` |
| `downloads` | `download_count DESC` |
| `rating` | `rating_avg × min(rating_count,50)/50 DESC` |
| `latest` | `updated_at DESC` |

> `norm(x) = (x - min) / (max - min + 1e-9)`，按当前数据集整体归一。
