## 1. 愿景与目标

| 03 痛点 | 04 目标 |
|---------|---------|
| 多文件 skill 无法上传 | 支持 **ZIP 压缩包发布**，后端解析 `skill.md` 自动入库 |
| 新手写不出规范 skill.md | 提供**在线创建页** + **LLM 草稿生成** |
| 关键词搜索理解力有限 | 提供 **LLM 智能搜索**，自然语言匹配 ≥ 5 条 |
| 识别不出"好 skill" | 展示 **评分 / 下载 / 收藏 / 点赞**；支持 **热度排序** |
| 夜间使用不友好 | 提供 **深色 / 浅色主题切换**，本地记忆偏好 |

## 2. 目标用户

| 角色 | 使用目标 | 频率 |
|------|----------|------|
| 技能消费者 | 检索技能 → 查看详情 → 下载 ZIP / 点赞 / 收藏 | 每周多次 |
| 技能作者 | ZIP 上传发布 或 在线（LLM 协助）创建 | 每月若干次 |

> v0.2 **不区分角色权限**，所有人可浏览、创建、上传、点赞、收藏、评分；互动幂等性通过浏览器 `visitor_id` 标识。

## 3. 核心业务场景

| 编号 | 场景名 | 用户路径 |
|------|--------|----------|
| SC-01 | **ZIP 发布** | 点击"上传 ZIP" → 选择文件 → 后端解压校验 `skill.md` → 成功后跳转详情页 |
| SC-02 | **在线创建（LLM 协助）** | 点击"新建 Skill" → 输入一句话意图 → 点"AI 生成草稿" → 编辑 → 保存 → 跳转详情页 |
| SC-03 | **浏览详情** | 列表页 → 点击卡片 → 查看 `skill.md` + 元数据 + 互动按钮 |
| SC-04 | **主题切换** | 点击 Header 右上角月亮/太阳图标 → 全站主题切换 → 刷新保留 |
| SC-05 | **分类 + 标签 + 排序筛选** | 侧栏选分类 → 顶部选标签芯片 → 下拉选择排序（hot/downloads/rating/latest） |
| SC-06 | **LLM 智能搜索** | 搜索框切换到"智能搜索"模式 → 输入自然语言 → 等待 ≤5 秒 → 返回 ≥5 条结果卡片 |
| SC-07 | **点赞 / 收藏** | 详情页或卡片上点击 👍 / ⭐ → 计数 +1 → 再次点击取消 |
| SC-08 | **评分** | 详情页星级组件点击 1~5 星 → 本次 `visitor_id` 评分覆盖 |

## 4. 产品规则

| 编号 | 规则 | 来源 |
|------|------|------|
| R-01 | ZIP 包必须包含根目录的 `skill.md`，否则返回 400 `MISSING_SKILL_MD` | 03 §3 |
| R-02 | ZIP 包大小 ≤ **10MB**，解压后总大小 ≤ **50MB**，文件数 ≤ **200** | 03 §5 风险 |
| R-03 | ZIP 解压路径受白名单约束（`./data/bundles/<skill_id>/`），禁止 `..` / 绝对路径 | 03 §5 风险 |
| R-04 | 在线创建必填字段：`name / category / description / skill_md` | SC-02 |
| R-05 | LLM 创建草稿接口允许 1 次/5 秒/`visitor_id` 限速 | 03 §5 风险 |
| R-06 | 智能搜索必须返回 ≥ 5 条结果（不足时用关键词兜底补齐） | 03 §4 |
| R-07 | 智能搜索超时或 LLM 不可用时**自动降级**为关键词搜索，并在响应中 `fallback=true` | 03 §5 |
| R-08 | 点赞 / 收藏 **幂等**：同一 `visitor_id` + `skill_id` 只能 +1 | 03 §4 |
| R-09 | 评分：同一 `visitor_id` 对同一 `skill_id` 的评分**覆盖**而非累加；`rating_avg` 为所有评分算术平均 | SC-08 |
| R-10 | 默认排序 `hot`：`0.5 × norm(download_count) + 0.3 × norm(rating_avg × rating_count) + 0.2 × norm(like_count + favorite_count)` | 03 §4 |
| R-11 | 主题切换：`light` / `dark` 二值，存 `localStorage.theme`，首次访问跟随系统偏好 | SC-04 |
| R-12 | `skill.md` 渲染经过 XSS 消毒，禁止 `<script>` / `on*` 事件 | 03 §5 |

## 6. 验收与成功标准（PRD 层）

| 标准 | 度量 |
|------|------|
| ZIP 可发布 | 10MB zip 在 30 秒内解析入库并跳转详情 |
| LLM 草稿可生成 | 输入一句话，10 秒内返回 ≥ 100 字符的 `skill.md` 草稿 |
| 智能搜索达标 | 对 10 条测试 query 均返回 ≥ 5 条 |
| 主题记忆 | 刷新与重开浏览器后主题保持一致 |
| 互动幂等 | 同一 visitor 对同一 skill 重复点赞/收藏后计数仍为 1 |
| 热度可重算 | 计数变化后 1 秒内下次 `sort=hot` 返回新顺序 |
