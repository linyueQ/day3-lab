## 1. 编号体系

| 编号格式 | 含义 | 示例 |
|----------|------|------|
| `REQ-M2HUB-{NNN}` | 功能需求 | `REQ-M2HUB-001` |
| `REQ-M2HUB-002a` | 子需求 | `REQ-M2HUB-002a` |
| `R-{NN}` | 产品规则（来自 `04`） | `R-01` |
| `TC-M02-{NNN}` | 测试用例 | `TC-M02-070` |

| 级别 | 含义 | 门禁影响 |
|------|------|----------|
| **P0** | 必须实现 | 关联 TC 全绿方可合并 |
| **P1** | 应当实现 | 未完成需说明 |
| **P2** | 可选 | 不纳入 v0.2 门禁 |

### 关键风险

| 风险项 | 严重度 | 说明 |
|--------|--------|------|
| ZIP 安全校验漏项 | **高** | 直接影响文件系统边界 |
| LLM fallback 缺失 | 高 | 会把可降级能力变成主链路故障 |
| 热度公式与排序实现不一致 | 中 | 影响默认体验与验收口径 |
| 互动聚合与明细漂移 | 中 | 计数不可信，收藏列表错乱 |
