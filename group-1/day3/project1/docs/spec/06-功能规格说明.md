## 2. Header 与主题

### 2.1 Header 元素

| 元素 | 行为 |
|------|------|
| Logo | 返回首页（清筛选） |
| 搜索框 | 支持两种模式：**关键词搜索**（默认，300ms 防抖）与 **智能搜索**（点击 🔮 图标切换，回车发起） |
| 新建按钮 | 跳转 `/create` |
| 上传 ZIP 按钮 | 跳转 `/upload` |
| 🌙 / ☀ 主题按钮 | 切换主题，详见 2.2 |

### 2.2 主题切换（对齐 US-004）

| 步骤 | 行为 |
|------|------|
| 初始化 | `theme = localStorage.theme ?? (matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light')` |
| 切换 | `document.documentElement.setAttribute('data-theme', newTheme)`；同步 `localStorage.theme` |
| 样式 | 所有颜色以 CSS 变量定义：`--bg / --fg / --card / --border / --primary / --muted`；深浅色在 `:root[data-theme=...]` 下覆写 |
| 对比度 | 文本 vs 背景 ≥ 4.5:1（WCAG AA） |
| 覆盖范围 | Header / Sidebar / Main / 卡片 / 代码块 / 模态框 / 表单元素全部切换 |

## 3. 列表页（SkillList）

### 3.1 API 调用

| 场景 | 请求 |
|------|------|
| 初次加载 | `GET /skills?sort=hot&page=1&page_size=12` |
| 分类 | `GET /skills?category=frontend` |
| 标签 | `GET /skills?tags=react,test` |
| 排序 | `sort=hot` / `downloads` / `rating` / `latest` |
| 关键词 | `GET /skills?q=...` |
| 智能搜索 | `POST /skills/smart-search { query }` |

### 3.2 三态渲染

| 状态 | 条件 | 显示 |
|------|------|------|
| 加载中 | `loading` | 骨架屏（12 卡片） |
| 空结果 | `items.length === 0 && !loading` | 空状态插画 + "清空筛选" |
| 正常 | `items.length > 0` | 卡片网格 + 分页 |

### 3.3 排序下拉

| 选项 | 后端 `sort` 值 | 说明 |
|------|---------------|------|
| 🔥 热度 | `hot` | 综合公式（04 R-10） |
| ⬇ 下载量 | `downloads` | `download_count` 倒序 |
| ⭐ 评分 | `rating` | 加权评分（05 AC-006-02） |
| 🆕 最新 | `latest` | `updated_at` 倒序 |

### 3.4 SkillCard 元素

| 元素 | 字段 |
|------|------|
| 标题 | `name` |
| 描述 | `description`（截断 2 行） |
| 分类 | `category` |
| 标签 | `tags[]`（最多显示 4 个） |
| 互动统计 | ⭐`rating_avg`(`rating_count`) / 👁`view_count` / ⬇`download_count` / 👍`like_count` / ⭐`favorite_count` |
| 快捷操作 | 卡片悬浮时显示"点赞"、"收藏" |

## 4. 搜索与过滤行为

### 4.1 关键词搜索（默认）

| 项 | 规格 |
|----|------|
| 触发 | 输入 300ms 防抖 |
| 范围 | `name / description / tags` |
| URL 同步 | `?q=xxx` |

### 4.2 智能搜索（对齐 US-007）

```
用户切到智能搜索模式 → 输入自然语言 → 回车
   ↓
POST /skills/smart-search { query }
   ↓
加载态：「AI 正在匹配… 预计 ≤10 秒」
   ↓
返回 items[]（≥5 条），每项展示 match_reason
   ↓
若 fallback=true，在顶部显示 "AI 暂不可用，已使用关键词兜底"
```

| 项 | 规格 |
|----|------|
| 输入长度 | 1–200 字符 |
| 超时 | 前端 Axios/fetch 设置 12 秒硬超时（后端 10 秒） |
| 空结果 | 永远不出现（后端保证 ≥5） |

### 4.3 标签芯片（US-005）

| 项 | 规格 |
|----|------|
| 数据源 | `GET /skills/tags` → 返回 `{ tag, count }` 数组 |
| 展示 | Top 20 高频标签，按频次降序 |
| 多选 | 已选高亮；再次点击取消；URL 同步 `?tags=a,b` |
| 过滤逻辑 | AND |

## 5. 详情页（SkillDetail）

### 5.1 数据加载

| 时机 | 请求 |
|------|------|
| 进入页面 | `GET /skills/<id>` |
| 副作用 | `view_count` 原子 +1 |
| 404 | 显示 "技能不存在" 占位页 |

### 5.2 互动栏（对齐 US-003 / US-009 / US-010 / US-011）

| 元素 | 行为 |
|------|------|
| **⬇ 下载 ZIP** | `window.open('/api/v1/hub/skills/<id>/download')`，同时前端自增 `download_count` 展示 |
| **👍 点赞按钮** | 未点赞 → `POST /like`；已点赞 → `DELETE /like`。按钮态切换；显示 `like_count` |
| **⭐ 收藏按钮** | 逻辑同点赞；`favorite_count` |
| **星级组件** | 1~5 星点击 → `POST /rate { score }`；显示 `rating_avg` 与 `rating_count` |
| **视图计数** | 只读展示 `view_count` |

**乐观更新**：点击后立即 UI 自增，API 失败时回滚并提示 Toast。

### 5.3 skill.md 渲染

| 项 | 规格 |
|----|------|
| 渲染链 | 后端 `marked` → `bleach` 消毒 → `skill_md_html`；前端 `dangerouslySetInnerHTML` 前经 `DOMPurify` 二次消毒 |
| 代码高亮 | `highlight.js` |
| 图片 | 允许 `<img src>`，但 src 必须为 http(s) 或相对路径，禁 `javascript:` |

## 6. 创建页（SkillCreate，对齐 US-002 / US-002a）

### 6.1 表单字段

| 字段 | 类型 | 校验 |
|------|------|------|
| name | text | 非空 1–80 |
| category | select | 白名单 |
| tags | chip input | 0–6 项 |
| description | textarea | 非空 1–200 |
| skill_md | textarea + 预览 | 非空 1–10000 |

### 6.2 LLM 协助

| 步骤 | 行为 |
|------|------|
| 1. 用户在 "💡 AI 协助" 输入框写 ≥ 10 字意图 | — |
| 2. 点"生成草稿" → Button 置 loading | 倒计时 10 秒 |
| 3. `POST /skills/draft { intent }` | Body `intent` ≤ 200 字符 |
| 4. 成功 → 将 `skill_md_draft` 填入 `skill_md` 文本域（覆盖需确认） | — |
| 5. 若 `fallback=true` → Toast "AI 暂不可用，已使用模板草稿" | — |
| 6. 限速：同 visitor_id 5 秒冷却 | 按钮显示倒计时 |

### 6.3 提交

`POST /skills` → 成功 201 → 跳转 `/skills/<skill_id>`。

## 7. 上传页（SkillUpload，对齐 US-001）

### 7.1 步骤

```
1. 选择/拖拽 zip
2. 前端校验 size ≤ 10MB、扩展名 = .zip
3. FormData 上传 → POST /skills/upload（multipart/form-data）
4. 进度条显示上传百分比
5. 后端解压校验 → 成功返回 201 + skill_id → 跳转详情
6. 失败显示 error.code 对应文案
```

### 7.2 错误文案映射

| `error.code` | 前端文案 |
|--------------|----------|
| `FILE_TOO_LARGE` | "压缩包超过 10MB，请精简后重试" |
| `MISSING_SKILL_MD` | "压缩包根目录必须包含 skill.md" |
| `BUNDLE_LIMIT_EXCEEDED` | "解压后文件数或总大小超限" |
| `UNSAFE_ZIP` | "压缩包包含不安全路径，已拒绝" |
| `UPSTREAM_ERROR` | "服务器开小差了，请稍后重试" |

## 8. 状态管理

| State | 类型 | 初始值 | 用途 |
|-------|------|--------|------|
| `theme` | `'light' \| 'dark'` | 系统偏好 | 主题 |
| `items` | `Skill[]` | `[]` | 列表 |
| `total / page / page_size` | `number` | — | 分页 |
| `filters` | `{category,tags,q,sort,mode}` | `{sort:'hot',mode:'keyword'}` | 筛选 |
| `loading / error` | `boolean / string\|null` | — | 状态 |
| `visitorId` | `string` | cookie | 互动身份 |
| `liked / favorited` | `Record<skillId, boolean>` | `{}` | 本访客已点赞/收藏集合 |

| `INVALID_CATEGORY` | 400 | "分类不合法" |
| `INVALID_TAG` | 400 | "标签不合法" |
| `INVALID_QUERY` | 400 | "查询过长" |
| `INVALID_RATING` | 400 | "评分必须 1–5" |

| `MISSING_SKILL_MD` | 400 | 7.2 文案 |
