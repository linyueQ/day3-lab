## 1. 性能需求

### 1.1 延迟 SLO

| 指标 | 目标值 | 度量方式 | 来源 |
|------|--------|----------|------|
| **黄金记录 CRUD P95 延迟** | < **500ms** | 服务端计时 | `03` §4 |
| **资产总览计算延迟** | < **1000ms** | 服务端计时 | `03` §4 "1 秒内更新" |
| **金价查询延迟** | < **2000ms** | 含缓存查询 | `03` §4 |
| **AI 识图响应延迟** | < **3000ms** | OCR 服务端到端 | `03` §4 |
| 页面首屏渲染 | < **2000ms** | Lighthouse 性能评分 | 用户体验 |

### 1.2 降级策略

| 场景 | 降级行为 | 说明 |
|------|----------|------|
| 金价 API 不可用 | 返回本地缓存数据 + `updated_at` 时间戳 | 用户可见"数据更新于 XX:XX" |
| AI 识图服务超时 | 返回 `weight: null` | 前端提示"识别失败，请手动输入" |
| 本地存储异常 | 返回 500 + 具体错误码 | 前端展示友好错误页 |

## 2. 可用性需求

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 服务可用性 | **99.5%** | 月度统计 |
| MTTR | < 10 分钟 | 重启 Flask 进程即可恢复 |
| 数据导出 | < 1 分钟 | 支持 JSON 格式导出备份 |

## 3. 可观测性需求

### 3.1 日志

| 项 | 规格 |
|----|------|
| 日志格式 | JSON 结构化日志 |
| 日志级别 | INFO / WARNING / ERROR |
| 关键日志 | 金价 API 调用、OCR 调用、数据库异常 |

### 3.2 错误码体系

| HTTP | error.code | 触发条件 |
|------|------------|----------|
| 400 | `40001` | 参数校验失败 |
| 400 | `40002` | 账本不存在 |
| 400 | `40003` | 记录不存在 |
| 400 | `40004` | 默认账本不可删除 |
| 500 | `50001` | 金价 API 调用失败 |
| 500 | `50002` | AI 识图服务异常 |
| 500 | `50003` | 服务器内部错误 |

## 4. 安全需求

| 项 | 当前实现 | 说明 |
|----|----------|------|
| API 认证 | 无需认证（本地应用） | 如需登录可对接 JWT |
| 数据传输 | HTTPS 强制 | 浏览器安全要求 |
| 速率限制 | 100 req/min/user | 防止接口滥用 |

### 4.1 敏感数据保护

| 数据类型 | 保护方式 |
|----------|----------|
| 第三方 API Key（金价/OCR） | 环境变量存储（`.env`），禁止硬编码 |
| 用户黄金资产数据 | 存储在浏览器 LocalStorage/IndexedDB，不上传到服务器 |
| 照片文件 | 存储在服务器本地或 OSS，访问需鉴权 |

## 5. 数据约束

### 5.1 输入约束

| 约束 | 规格 | 来源 |
|------|------|------|
| `weight` 最小值 | 0.01 克 | `04` R-02 |
| `weight` 最大值 | 99999.99 克 | `04` R-02 |
| `total_price` 最小值 | 0.01 元 | `04` R-03 |
| `channel` 最大长度 | 50 字符 | `09` §3 |
| `note` 最大长度 | 200 字符 | `09` §3 |
| `ledger.name` 长度 | 1~20 字符 | `09` §8 |
| `photos` 最大数量 | 3 张 | `04` R-07 |
| `photos` 单张大小 | ≤ 2MB | `04` R-07 |

### 5.2 类型约束

| 字段 | 类型 | 约束 |
|------|------|------|
| `weight` | `DECIMAL(10,2)` | 精确到 2 位小数 |
| `total_price` | `DECIMAL(12,2)` | 精确到 2 位小数 |
| `mode` | `string` | 枚举：summary \| detail |
| `type`（赠卖） | `string` | 枚举：gift \| sell |
| `is_default` | `boolean` | MUST 为 true/false |
