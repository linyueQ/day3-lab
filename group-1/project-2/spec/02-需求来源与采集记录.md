## S-02 用户访谈 — 黄金投资爱好者

| 项 | 内容 |
|---|---|
| 访谈对象 | 张女士（黄金投资爱好者，3年经验） |
| 访谈人 | 产品经理 |
| 时间 | 2026-04-10 14:00-15:00 |
| 方式 | 线上访谈 |

| 问题 | 回答要点 |
|------|----------|
| Q1：平时怎么记录黄金购买？ | 用 Excel 记，但经常忘记更新，渠道多了就乱了。每次要算总重量和收益都很麻烦。 |
| Q2：最关心黄金的哪些数据？ | ① 实时金价 → ② 总持仓重量和当前价值 → ③ 整体收益率 |
| Q3：购买黄金的渠道有哪些？ | 银行金条、周大福等金店、支付宝黄金、京东金条，渠道很分散。 |
| Q4：对记账工具有什么期望？ | 最好能拍照自动识别重量，自动关联当天金价，一键看收益。操作要简单，最好小程序就能用。 |

**总结：** ① 统一记录 + 自动计算为核心诉求 ② 多渠道黄金统一管理 ③ 实时金价关联资产价值 ④ 小程序形态，随时随地记录
