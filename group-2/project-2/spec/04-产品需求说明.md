## 1. 愿景与目标

> 把 `03` §1 的痛点翻译成正面目标。

| 03 痛点 | 04 目标 |
|---------|---------|
| 基金健康状况缺乏一站式诊断入口 | 构建一站式基金智能诊断平台，用户输入代码即可获取多维评分与投资建议 |
| 持仓股利好/利空新闻分散在不同平台 | 自动聚合持仓股新闻资讯，通过 NLP 情感分析标注利好/利空/中性 |
| 基金行情数据分散，筛选排序操作繁琐 | 提供统一行情浏览入口，支持多维筛选、排序与收藏 |
| 行业资讯阅读体验差 | 构建分类资讯阅读平台，支持搜索、互动与个性化推荐 |
| 缺少便捷的智能问答入口 | 接入真实 LLM 大模型，提供全局悬浮问答助手，支持多轮上下文记忆 |
| 自选基金管理功能缺失 | 支持网页端自选基金管理与持仓收益跟踪 |

## 2. 目标用户

| 角色 | 使用目标 | 频率 |
|------|----------|------|
| 个人投资者 | 诊断已持有基金、浏览行情、获取买卖建议 | 每天多次 |
| 基金研究员 | 跟踪行业资讯、分析持仓股新闻、对比基金表现 | 每天多次 |
| 理财顾问 | 为客户筛选基金、生成诊断报告、回答客户疑问 | 每周多次 |

## 3. 核心业务场景

| 编号 | 场景名 | 用户路径 |
|------|--------|----------|
| SC-01 | 基金诊断全链路 | 用户输入基金代码 → 查看综合评级 + 5维评分 → 浏览历史收益/持仓 → 查看持仓股利好利空资讯 → 获取投资建议 |
| SC-02 | 基金行情浏览 | 用户进入行情页 → 查看大盘指数 → 按类型筛选/搜索基金 → 按涨跌排序 → 收藏感兴趣的基金 |
| SC-03 | 资讯阅读与互动 | 用户进入视点页 → 浏览头条文章 → 按分类/标签筛选 → 阅读详情 → 点赞 |
| SC-04 | AI 问答获取建议 | 用户在任意页面点击悬浮按钮 → 输入"白酒基金还能买吗" → LLM 返回通俗建议 → 继续追问 |
| SC-05 | 自选基金管理 | 用户进入自选页 → 查看持仓汇总 → 添加新基金到自选 → 删除不再关注的基金 |

## 4. 产品规则

| 编号 | 规则 | 来源 |
|------|------|------|
| R-01 | 基金诊断 MUST 返回综合评级（1-5星）和5维度评分（0-100分） | 03 §3 范围1 |
| R-02 | 持仓股资讯 MUST 通过 NLP 标注情感倾向（positive/negative/neutral） | 03 §3 范围2 |
| R-03 | 基金行情列表 MUST 支持按类型筛选 + 关键词搜索 + 多列排序 | 03 §3 范围3 |
| R-04 | 基金问答 MUST 接入真实 LLM，1.5 秒内返回真实回复 | 03 §4 成功标准3 |
| R-05 | 基金问答 MUST 支持多轮上下文记忆，同一会话内可连续追问 | 03 §3 范围5 |
| R-06 | 自选基金 MUST 展示持仓汇总（总市值/成本/收益/收益率） | 03 §3 范围6 |
| R-07 | 所有涨跌幅展示 MUST 遵循红涨绿跌配色规范 | 02 S-03 交互友好 |
| R-08 | 前端 MUST 适配移动端响应式布局 | 03 §3 范围7 |

## 5. 能力与优先级

| 子能力 | 优先级 | 对应场景 |
|--------|--------|----------|
| 基金诊断（搜索 + 综合评级 + 5维评分） | **P0** | SC-01 |
| 基金诊断（历史收益 + 前十大持仓 + 投资建议） | **P0** | SC-01 |
| 持仓股资讯聚合（NLP 情感分析 + 利好利空标注） | **P0** | SC-01 |
| 基金问答（真实 LLM + 多轮上下文 + 悬浮窗口） | **P0** | SC-04 |
| 基金行情（大盘指数 + 基金列表 + 筛选排序收藏） | **P1** | SC-02 |
| 基金视点（资讯浏览 + 分类搜索 + 点赞 + 标签云） | **P1** | SC-03 |
| 自选基金管理（添加/删除 + 持仓收益汇总） | **P1** | SC-05 |
| 全局导航（侧边栏 + 响应式 + 移动端适配） | **P1** | 全部 |

## 6. 验收与成功标准（PRD 层）

| 标准 | 度量 |
|------|------|
| 诊断可演示 | 输入基金代码后 30 秒内完成"搜索 → 评级 → 持仓资讯 → 投资建议"全链路 |
| 问答可演示 | 用户提问后 1.5 秒内收到真实 LLM 回复，支持连续追问 |
| 行情可演示 | 类型筛选 + 搜索 + 排序操作 10 秒内定位目标基金 |
| 自选可演示 | "添加 → 查看汇总 → 删除"全流程可走通 |
| 移动端可用 | 在 375px 宽度设备上所有功能可正常使用 |
| 零错误 | 切换全部页面 + 核心操作后，控制台无 Error 级日志 |
