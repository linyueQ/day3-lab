## 2. 全局导航（对齐 US-008）

### 2.1 侧边栏 Sider

| 元素 | 行为 |
|------|------|
| Logo + "基金管家" | 展开时显示文字，折叠时仅显示图标 |
| 菜单项 × 4 | 基金诊断、基金行情、基金视点、我的自选 |
| 当前页高亮 | 根据 `location.pathname` 自动匹配 |
| 折叠按钮 | 点击切换展开/折叠，宽度 200px ↔ 80px |

### 2.2 顶部栏 Header

| 元素 | 行为 |
|------|------|
| 折叠触发器 | `MenuFoldOutlined` / `MenuUnfoldOutlined` 切换 |
| 通知铃铛 | `Badge` 显示未读数量角标 |
| 用户头像 | 右侧固定显示 |

## 3. 基金诊断页（对齐 US-001 / US-002 / US-003）

### 3.1 搜索区域

| 元素 | 行为 |
|------|------|
| 渐变背景区 | 线性渐变蓝色背景，居中标题"基金智能诊断" |
| 搜索框 | `Input.Search`，placeholder "输入基金代码或名称" |
| 热门标签 | 6个预设基金标签，点击自动填入代码并触发搜索 |

### 3.2 诊断结果 — 综合评级（US-001）

| 元素 | 行为 |
|------|------|
| 基金信息头 | 展示名称、代码、类型、基金经理、公司、规模 |
| 综合评级 | 1-5 星 `Rate` 组件 + 评级标签 Tag |
| 最新净值 | 净值数字 + 日涨跌幅（红涨绿跌） |
| 5维评分仪表盘 | 5 个 `Progress` 圆形进度条：收益/风控/稳定性/择时/选股 |

### 3.3 诊断详情 Tab（US-002）

| Tab | 内容 |
|-----|------|
| 历史收益 | 6 个时段收益率卡片，正值红色带"+"，负值绿色 |
| 前十大持仓 | 表格：股票名称、代码、行业 Tag、占比 Progress |
| 投资建议 | 诊断结论 Alert + 建议列表（带序号图标） |

### 3.4 持仓股资讯区域（US-003）

| 元素 | 行为 |
|------|------|
| 区域标题 | "持仓股近期资讯" + `NotificationOutlined` 图标 |
| 双列卡片布局 | `Row` + `Col span={12}` |
| 卡片内容 | 情感 Tag + 股票名称代码 + 标题 + 来源 + 时间 |
| 情感配色 | 利好：红色左边框+浅红背景；利空：绿色+浅绿；中性：灰色+浅灰 |
| 排序 | 按 `published_at` 倒序 |

## 4. 基金行情页（对齐 US-004）

### 4.1 大盘指数

| 元素 | 行为 |
|------|------|
| 4 个指数卡片 | 上证指数、深证成指、创业板指、沪深300 |
| 卡片内容 | 指数值 + 涨跌幅 + 成交量 |
| 顶部色条 | 涨红跌绿，3px 圆角顶部边框 |

### 4.2 基金列表

| 功能 | 实现 |
|------|------|
| 类型筛选 | `Segmented` 组件：全部/混合型/指数型 |
| 关键词搜索 | `Input.Search`，按名称或代码过滤 |
| 表格列 | 名称、代码、类型、净值、日涨跌、周、月、年、规模、风险 |
| 排序 | 日涨跌、近1周、近1月、近1年列可升降排序 |
| 收藏 | 星标图标，点击切换收藏状态 |
| 涨跌配色 | 正值红色，负值绿色，零值灰色 |

## 5. 基金视点页（对齐 US-005）

### 5.1 头条区域

| 元素 | 行为 |
|------|------|
| 头条大图卡片 | 第一篇文章的封面图 + 标题 + 摘要 + 来源 + 阅读量 |
| 分类 Tag 条 | 全部/市场展望/行业动态/投资策略/产品分析，点击筛选 |
| 搜索框 | 按标题/摘要/标签搜索 |

### 5.2 文章列表

| 元素 | 行为 |
|------|------|
| 文章卡片 | 分类 Tag + 标题 + 摘要 + 作者头像+名称 + 时间 + 阅读量 |
| 点赞 | `HeartOutlined` / `HeartFilled`，点击切换，计数实时更新 |

### 5.3 右侧栏

| 元素 | 行为 |
|------|------|
| 热门排行 | 前 5 篇文章按阅读量排序，带序号色标（金银铜） |
| 标签云 | 热门标签列表，点击触发搜索 |

## 6. 自选基金页（对齐 US-007）

### 6.1 持仓汇总

| 元素 | 行为 |
|------|------|
| 4 个统计卡片 | 总市值、总投入成本、持仓总收益（正红负绿）、总收益率 |
| 渐变背景 | 蓝色渐变背景区 |

### 6.2 自选列表

| 元素 | 行为 |
|------|------|
| 表格列 | 星标、名称代码、类型、净值、日/周/月涨跌、持有收益、添加日期 |
| 添加按钮 | 点击弹出 Modal，输入代码/名称搜索，点击"添加" |
| 重复检测 | 添加已有基金时 `message.warning("该基金已在自选列表中")` |
| 删除 | 点击删除图标弹出 `Modal.confirm`，确认后移除 |
| 空状态 | `Empty` 组件 + "添加您关注的基金" 提示 + 快捷添加按钮 |

## 7. 基金问答悬浮窗（对齐 US-006）

| 元素 | 行为 |
|------|------|
| 悬浮按钮 | 右下角 fixed，蓝色圆形 56px，`MessageOutlined` + "问一问" |
| 聊天窗口 | 400×560px，展开动画 |
| 欢迎语 | "你好！我是基金智投助手..." + 快捷提问标签 |
| 用户消息 | 蓝色气泡，右对齐 |
| AI 回复 | 白色气泡，左对齐，0.8-2s 延迟，"思考中..."加载态 |
| 发送按钮 | 输入为空或 AI 回复中时 disabled |
| 自动滚动 | 新消息自动滚动到底部 |
| 关闭 | 右上角 × 按钮收起窗口 |

## 8. 错误处理

| 后端 `error.code` | HTTP | 前端展示 |
|--------------------|------|----------|
| `EMPTY_QUERY` | 400 | `message.error("请输入问题")` |
| `INVALID_QUERY` | 400 | `message.error("问题过长，请控制在500字以内")` |
| `INVALID_FUND_CODE` | 400 | `message.error("基金代码格式不正确")` |
| `FUND_NOT_FOUND` | 404 | `message.warning("未找到该基金")` |
| `DUPLICATE_WATCHLIST` | 409 | `message.warning("该基金已在自选列表中")` |
| `LLM_ERROR` | 500 | 聊天窗口显示"暂时无法回答，请稍后重试" |
| 网络错误 | — | `message.error("网络异常，请检查连接")` |

## 9. 状态管理

| State 变量 | 类型 | 初始值 | 所在组件 | 用途 |
|------------|------|--------|----------|------|
| `collapsed` | `boolean` | `false` | AppLayout | 侧边栏折叠状态 |
| `searchResult` | `Fund\|null` | `null` | DiagnosisPage | 诊断结果 |
| `activeTab` | `string` | `'returns'` | DiagnosisPage | 详情 Tab |
| `marketFunds` | `Fund[]` | `[]` | MarketPage | 行情基金列表 |
| `filterType` | `string` | `'all'` | MarketPage | 类型筛选 |
| `articles` | `Article[]` | `[]` | InsightsPage | 文章列表 |
| `category` | `string` | `'all'` | InsightsPage | 分类筛选 |
| `watchlist` | `WatchlistItem[]` | `[]` | WatchlistPage | 自选基金 |
| `chatOpen` | `boolean` | `false` | FundChatBot | 聊天窗口开关 |
| `messages` | `Message[]` | `[]` | FundChatBot | 消息列表 |
| `loading` | `boolean` | `false` | FundChatBot | AI 回复中 |
