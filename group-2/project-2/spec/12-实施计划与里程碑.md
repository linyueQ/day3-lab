## 5. 依赖与风险

**依赖链**：

**S1 前端原型** → **S2 后端API**（需 API 才能对接真实数据）→ **S3 LLM+NLP**（需后端基础设施）→ **S4 收口**

| 风险 | 影响 | 应对 |
|------|------|------|
| 真实基金数据 API 接入延迟 | S2 延期 | Mock API → 渐进式替换 |
| LLM 大模型服务不稳定 | 问答体验下降 | 三级降级策略，兜底模式始终可用 |
| NLP 情感分析准确率不达标 | 错误标注影响用户决策 | 人工抽检 + 模型迭代 + 置信度阈值 |
| 移动端适配工作量超预期 | S4 延期 | 优先适配核心页面（诊断+问答） |
| Ant Design 6 升级 breaking changes | 页面样式异常 | 锁定 antd@6.x 版本 |
