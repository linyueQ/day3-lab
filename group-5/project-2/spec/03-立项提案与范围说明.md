## 1. 项目背景

### 1.1 市场背景

命理测算在中国拥有深厚的文化底蕴和广泛的群众基础。随着现代生活节奏加快，人们在面临重要决策时，往往希望通过传统文化中的命理智慧获取参考和指引。市场调研显示：

- **用户需求旺盛**: 星座、八字、塔罗等命理相关内容在社交媒体和搜索平台持续高热
- **数字化趋势**: 传统命理服务正加速向线上迁移，移动端和Web端应用需求增长迅速
- **个性化需求**: 用户不再满足于通用的命理内容，更渴望基于个人八字的专属测算服务

### 1.2 业务痛点

当前市场上的命理测算产品存在以下问题：

1. **同质化严重**: 大多数产品提供通用黄历和标准化解读，缺乏个性化
2. **场景覆盖不足**: 未能覆盖用户从日常参考到特定事件决策的完整需求链
3. **用户体验欠佳**: 界面设计老旧，交互流程复杂，不符合现代Web应用标准
4. **算法透明度低**: 多数产品黑盒运行，用户无法理解测算逻辑

### 1.3 解决方案

开发一款**多功能命理工具集合Web应用**，聚焦三大核心场景：

- **个性化黄历日历**: 基于用户八字生成每日专属运势参考
- **针对性运势问询**: 支持奇门遁甲和大小六壬起卦，为特定事件提供决策参考
- **静态命格解析**: 深度解析个人命格，提供长期人生规划参考

## 3. 项目范围

### 3.1 包含范围（In-Scope）

#### 核心功能

| 功能模块 | 功能描述 | 优先级 |
|---------|---------|--------|
| 用户系统 | 用户注册、登录、个人档案管理（八字信息录入） | P0 |
| 个性化黄历日历 | 基于用户八字生成每日适宜/不适宜事项、幸运颜色、有利方向 | P0 |
| 奇门遁甲起卦 | 依托当前时间+空间信息自动起卦，单日单次测算 | P0 |
| 大小六壬起卦 | 用户手动输入3个数字起卦，支持单日多次测算 | P0 |
| 命格解析 | 输入姓名、出生日期、时辰，生成命格关键词和深度解析 | P0 |
| 响应式设计 | 支持PC、平板、移动端自适应浏览 | P0 |

#### 技术能力

| 能力模块 | 说明 |
|---------|------|
| 八字排盘引擎 | 支持公历转农历、天干地支计算、五行分析 |
| 奇门遁甲引擎 | 支持时家奇门起局、九宫排盘、吉凶判断 |
| 大小六壬引擎 | 支持月将加时、三传四课计算 |
| 命格解析引擎 | 基于规则的命格关键词匹配和解析内容生成 |

### 3.2 排除范围（Out-of-Scope）

| 功能/能力 | 排除原因 | 未来规划 |
|----------|---------|---------|
| 公开事件命理分析 | 与个人用户核心需求无关，属于娱乐性质 | MVP验证后评估 |
| 社交分享功能 | MVP聚焦核心测算能力 | V2.0迭代 |
| 付费订阅系统 | MVP阶段免费验证需求 | V2.0迭代 |
| AI对话式测算 | 需要额外的AI模型集成 | V3.0迭代 |
| 原生APP (iOS/Android) | Web端优先验证需求 | V2.0评估 |
| 微信小程序版本 | Web端优先，后续考虑多渠道 | V2.0评估 |

### 3.3 边界条件

#### 支持的范围
- **用户群体**: 中国大陆用户（简体中文）
- **浏览器支持**: Chrome 90+, Safari 14+, Firefox 88+, Edge 90+
- **设备支持**: 支持iOS 13+、Android 8+的移动端设备，以及主流PC浏览器
- **测算范围**: 1900年1月1日至2100年12月31日的日期范围

#### 不支持的范围
- 不支持非中文语言
- 不支持IE11及以下浏览器
- 不支持超出日期范围的历史人物命格解析
- 不提供医疗、法律等专业领域的决策建议
