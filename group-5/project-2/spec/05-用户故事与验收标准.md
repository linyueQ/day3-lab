## US-001: 用户注册与登录

**作为** 新用户  
**我想要** 通过手机号或邮箱注册账号并登录系统  
**以便于** 使用命理测算功能并保存我的个人数据

### 验收标准

- [ ] **正常注册流程**
  - Given 用户访问注册页面
  - When 输入有效的手机号/邮箱、验证码和密码
  - Then 系统创建账号并引导填写基础信息

- [ ] **密码强度校验**
  - Given 用户设置密码
  - When 密码长度小于8位或不包含字母和数字
  - Then 系统显示密码强度不足提示

- [ ] **登录成功**
  - Given 用户已注册账号
  - When 输入正确的账号和密码
  - Then 系统发放JWT Token并跳转到首页

- [ ] **登录失败**
  - Given 用户输入错误的账号或密码
  - When 点击登录按钮
  - Then 系统显示"账号或密码错误"提示

- [ ] **Token过期处理**
  - Given 用户Token已过期（24小时）
  - When 用户访问需要认证的页面
  - Then 系统自动跳转到登录页

### 优先级: P0
### 估算: 3天

## US-002: 个人档案管理

**作为** 已注册用户  
**我想要** 录入和编辑我的出生信息（姓名、性别、出生日期、出生时辰）  
**以便于** 系统能够基于我的八字提供个性化测算服务

### 验收标准

- [ ] **录入档案**
  - Given 用户首次登录系统
  - When 进入"我的档案"页面并填写完整信息
  - Then 系统自动计算八字并保存档案

- [ ] **公历转农历**
  - Given 用户输入公历出生日期
  - When 系统计算八字
  - Then 准确转换为农历并计算天干地支

- [ ] **时辰选择**
  - Given 用户不清楚具体出生时辰
  - When 选择"不清楚时辰"选项
  - Then 系统仅使用年月日三柱计算，并标注"精度降低"

- [ ] **修改档案**
  - Given 用户已有档案
  - When 修改出生信息并保存
  - Then 系统重新计算八字并更新档案

- [ ] **八字准确性**
  - Given 用户输入准确的出生信息
  - When 系统计算八字
  - Then 四柱（年月日时）计算准确率100%

### 优先级: P0
### 估算: 3天

## US-003: 查看个性化黄历

**作为** 已登录用户  
**我想要** 查看当日的个性化黄历（适宜/不适宜事项、幸运颜色、有利方向）  
**以便于** 在出门前快速了解当日运势参考

### 验收标准

- [ ] **首页展示**
  - Given 用户登录系统
  - When 访问首页
  - Then 自动展示今日个性化黄历，无需额外操作

- [ ] **个性化内容**
  - Given 用户A和用户B的八字不同
  - When 两人查看同日黄历
  - Then 显示的适宜/不适宜事项、幸运颜色等内容不同

- [ ] **内容完整性**
  - Given 用户查看黄历
  - When 页面加载完成
  - Then 显示3-5条适宜事项、2-3条不适宜事项、1-2种幸运颜色、1-2个有利方向

- [ ] **翻看历史**
  - Given 用户查看今日黄历
  - When 点击左右箭头或滑动屏幕
  - Then 可查看前后7天的黄历内容

- [ ] **缓存机制**
  - Given 用户查看今日黄历
  - When 24小时内再次查看
  - Then 系统从缓存读取，无需重新计算

### 优先级: P0
### 估算: 5天

- [ ] **输入事项**

  - When 选择"奇门遁甲"并输入要测算的事项

- [ ] **位置授权**

  - Then 3秒内返回结果

- [ ] **结果展示**
  - Given 起卦计算完成
  - When 展示结果页面
  - Then 显示卦象信息、事项建议（适合/不适合）、成功率百分比、详细解析（200-500字）

- [ ] **单日限制**
  - Given 用户今日已使用奇门遁甲起卦1次
  - When 尝试再次起卦
  - Then 系统提示"今日仅限1次，请明日再试或使用大小六壬"

- [ ] **卦象可视化**
  - Given 用户查看结果
  - When 页面渲染完成
  - Then 以九宫格形式可视化展示卦象
