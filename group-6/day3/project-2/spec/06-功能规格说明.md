## 1. 页面总体布局

```
┌─────────────────────────────────────────────────────────┐
│  header: 投研分析平台 + [导航标签页]                       │
│  [研报管理] [知识库] [研报比对]                              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  main-content（根据导航标签页切换）                        │
│                                                         │
│  A) 研报管理页 — 上传 + 列表 + 筛选                       │
│  B) 知识库页 — 股票列表 + 详情（研报+观点+行情）            │
│  C) 研报比对页 — 选择研报 + 比对结果                       │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 2. Header 导航区域

| 元素 | 类型 | 行为 |
|------|------|------|
| 平台标题 | 文本 | 显示"投研分析平台" |
| 导航标签页 | Tab 组件 | 3 个标签：研报管理、知识库、研报比对 |
| 当前标签 | 高亮样式 | 当前激活标签高亮显示 |

## 3. 研报管理页

> 对齐 `05` US-001（研报上传）、US-006（研报管理）。

### 3.1 上传区域

| 功能 | 触发 | API 调用 | 行为 |
|------|------|----------|------|
| 文件选择 | 点击"上传研报"按钮或拖拽 | — | 打开文件选择器，仅允许 .pdf |
| 上传文件 | 选择文件后自动上传 | `POST /api/v1/reports/upload` | 显示上传进度 → 成功后自动触发解析 |
| 自动解析 | 上传成功后 | `POST /api/v1/reports/{id}/parse` | 显示"解析中..." → 完成后刷新列表 |

### 3.2 研报列表

| 功能 | 触发 | API 调用 | 行为 |
|------|------|----------|------|
| 加载列表 | 页面渲染 | `GET /api/v1/reports` | 渲染研报卡片列表 |
| 筛选 | 输入股票代码/行业/时间 | `GET /api/v1/reports?stock_code=&industry=&date_from=&date_to=` | 实时过滤列表 |
| 查看详情 | 点击研报卡片 | — | 展开解析结果（标题、评级、目标价、核心观点、股票信息） |
| 删除 | 点击删除按钮 | `DELETE /api/v1/reports/{id}` | 确认弹窗 → 从列表移除 → 级联删除解析数据 |
| 下载原文 | 点击下载按钮 | `GET /api/v1/reports/{id}/file` | 下载原始 PDF 文件 |

### 3.3 研报卡片展示

| 字段 | 展示方式 |
|------|----------|
| 标题 | 卡片标题，加粗 |
| 股票代码 + 名称 | 标签样式，蓝色 |
| 行业 | 标签样式，灰色 |
| 评级 | 彩色标签（买入=绿、增持=浅绿、中性=灰、减持=橙、卖出=红） |
| 目标价 | 数值文本 |
| 上传时间 | 相对时间（"3小时前"）+ tooltip 显示完整时间 |
| 解析状态 | 状态标签（待解析/解析中/已完成/失败） |

## 4. 解析结果详情展示

> 对齐 `05` US-002（研报解析）。

| 字段 | 展示方式 |
|------|----------|
| 标题 | 大字号加粗 |
| 评级 | 彩色标签 + 文字说明 |
| 目标价 | 数值，单位：元 |
| 核心观点 | 多行文本，支持折叠/展开 |
| 股票代码 | 链接样式，点击跳转到知识库对应股票页面 |
| 股票名称 | 跟随股票代码显示 |
| 解析时间 | 完整时间戳 |
| 原文链接 | 点击下载/预览 PDF |

## 5. 知识库页面

> 对齐 `05` US-003（知识库浏览）、US-007（行情数据）。

### 5.1 股票列表视图

| 功能 | 触发 | API 调用 | 行为 |
|------|------|----------|------|
| 加载股票列表 | 页面渲染 | `GET /api/v1/kb/stocks` | 展示按股票聚合的列表，每项显示股票代码、名称、关联研报数量 |
| 搜索 | 输入股票代码或名称 | 前端过滤 | 实时过滤列表 |
| 进入详情 | 点击股票项 | `GET /api/v1/kb/stocks/{code}` | 展示该股票的详情页 |

### 5.2 股票详情视图

| 区域 | 内容 | API |
|------|------|-----|
| 股票概要 | 股票代码、名称、行业 | `GET /api/v1/kb/stocks/{code}` |
| 行情数据 | PE、PB、市值、最新价等 | `GET /api/v1/stocks/{code}/market-data` |
| 最近观点汇总 | 自动聚合多份研报的核心观点，按时间排序 | `GET /api/v1/kb/stocks/{code}` 内含 |
| 关联研报列表 | 该股票的所有研报，支持时间筛选/排序 | `GET /api/v1/kb/stocks/{code}/reports` |

### 5.3 行情数据区域

| 指标 | 说明 | 降级处理 |
|------|------|----------|
| PE（市盈率） | 最新数据 | AKShare 不可用时显示"暂无数据" |
| PB（市净率） | 最新数据 | 同上 |
| 市值 | 总市值 | 同上 |
| 最新价 | 最新收盘价 | 同上 |
| 数据时间 | 数据更新时间戳 | — |

## 6. 研报比对页

> 对齐 `05` US-004（研报比对）。

### 6.1 研报选择区域

| 功能 | 触发 | API 调用 | 行为 |
|------|------|----------|------|
| 选择股票 | 下拉选择或搜索 | `GET /api/v1/kb/stocks` | 加载该股票下的研报列表 |
| 选择研报 | 勾选 ≥ 2 份研报 | — | 勾选后启用"开始比对"按钮 |
| 发起比对 | 点击"开始比对" | `POST /api/v1/reports/compare` | 显示 loading → 展示比对结果 |

### 6.2 比对结果展示

| 区域 | 展示方式 |
|------|----------|
| 基本信息对照表 | 表格形式，每列一份研报，行：标题、评级、目标价 |
| 相似观点合并 | 合并相似的核心观点，以统一描述展示，标注来源研报 |
| 差异高亮 | 不同之处以黄色/红色高亮标注，差异列对比展示 |
| 评级差异 | 不同评级用不同颜色标签展示 |
| 目标价差异 | 数值不同时高亮，并显示差值/差距百分比 |

## 7. 错误处理

| 后端 `error.code` | HTTP | 前端展示 |
|--------------------|------|----------|
| `INVALID_FILE_TYPE` | 400 | "仅支持 PDF 格式文件" |
| `FILE_TOO_LARGE` | 400 | "文件大小不能超过 50MB" |
| `PARSE_FAILED` | 500 | "研报解析失败，请检查文件格式" |
| `REPORT_NOT_FOUND` | 404 | "研报不存在或已被删除" |
| `STOCK_NOT_FOUND` | 404 | "未找到该股票的知识库数据" |
| `COMPARE_MIN_REPORTS` | 400 | "至少选择 2 份研报进行比对" |
| `LLM_ERROR` | 500 | "AI 服务暂时不可用，请稍后重试" |

## 8. 状态管理

| State 变量 | 类型 | 初始值 | 用途 |
|------------|------|--------|------|
| `activeTab` | `string` | `'reports'` | 当前导航标签 |
| `reports` | `Report[]` | `[]` | 研报列表 |
| `selectedReport` | `Report\|null` | `null` | 当前查看的研报详情 |
| `stocks` | `Stock[]` | `[]` | 知识库股票列表 |
| `selectedStock` | `Stock\|null` | `null` | 当前查看的股票 |
| `marketData` | `MarketData\|null` | `null` | 当前股票行情数据 |
| `compareReports` | `string[]` | `[]` | 待比对的研报 ID 列表 |
| `compareResult` | `CompareResult\|null` | `null` | 比对结果 |
| `loading` | `boolean` | `false` | 请求是否进行中 |
| `error` | `string\|null` | `null` | 错误提示信息 |
| `uploadProgress` | `number` | `0` | 上传进度百分比 |
| `filters` | `object` | `{}` | 筛选条件 |
