## 3. Sidebar 会话管理

> 对齐 `05` US-001。描述会话列表的增删查交互。

| 功能 | 触发 | API 调用 | 行为 |
|------|------|----------|------|
| 加载列表 | 页面渲染 | `GET /sessions` | 渲染会话列表 + 默认选中第一个 |
| 新建会话 | 点击"+ 新建" | `POST /sessions` | 插入列表头部 + 自动选中 |
| 删除会话 | 点击 × | `DELETE /sessions/<id>` | 确认弹窗 → 从列表移除 |
| 选中会话 | 点击会话项 | `GET /chat/sessions/{session_id}/messages` | 1. 高亮选中会话项<br>2. 加载该会话的消息历史<br>3. 右侧显示关联研报信息<br>4. 输入框重置为空 |

## 4. Main 内容区（三态渲染）

> 对齐 `05` US-002（问答提交）、US-003（历史记录）。描述三种条件下显示什么。

| 状态 | 条件 | 显示内容 |
|------|------|----------|
| **A 空状态** | `currentReport === null` | "请上传或选择一篇研报开始" + 上传引导区 |
| **B 常见问题网格** | 有研报 && `messages.length === 0` | 展示预设常见问题卡片网格：<br>• "这家公司的核心竞争优势是什么？"<br>• "研报给出的目标价和评级是什么？"<br>• "公司未来三年的盈利预测如何？"<br>• "有哪些风险提示需要关注？"<br>点击卡片自动填入输入框并发送 |
| **C 对话历史卡片元素** | `messages.length > 0` | 消息列表，每条消息包含：<br>• **用户消息**：头像 + 问题内容<br>• **AI消息**：头像 + 答案内容 + 引用来源标签（4.1节）<br>• **引用展开**：点击引用编号展开原文片段卡片<br>• **时间戳**：消息发送时间<br>• **加载状态**：AI回复时显示打字动画 |

### 4.1 记录卡片中的来源标签逻辑

## 6. 错误处理

| 后端 `error.code` | HTTP | 前端展示 |
|--------------------|------|----------|
| `EMPTY_QUERY` | 400 | 错误提示："请输入问题" |
| `INVALID_QUERY` | 400 | 错误提示："问题过长，请控制在500字以内" |
| `QUERY_TOO_SHORT` | 400 | 错误提示："问题过短，请详细描述您的问题" |
| `REPORT_NOT_FOUND` | 404 | 错误提示："研报不存在或已删除" |
| `SESSION_NOT_FOUND` | 404 | 错误提示："会话不存在或已过期" |
| `UPLOAD_FAILED` | 422 | 错误提示："文件上传失败，请重试" |
| `FILE_TOO_LARGE` | 413 | 错误提示："文件大小超过50MB限制" |
| `INVALID_FILE_TYPE` | 415 | 错误提示："仅支持PDF和HTML格式文件" |
| `PARSE_FAILED` | 422 | 错误提示："研报解析失败，请检查文件格式" |
| `PARSE_TIMEOUT` | 504 | 错误提示："解析超时，请稍后重试或联系管理员" |
| `STOCK_NOT_FOUND` | 404 | 错误提示："未找到该股票信息" |
| `STOCK_DATA_ERROR` | 503 | 错误提示："股票数据加载失败，请刷新重试" |
| `COMPARE_LIMIT_EXCEEDED` | 422 | 错误提示："对比研报数量超过限制（最多5篇）" |
| `COMPARE_SAME_COMPANY_ONLY` | 422 | 错误提示："仅支持同一公司的研报对比" |
| `RATE_LIMIT_EXCEEDED` | 429 | 错误提示："请求过于频繁，请稍后再试" |
| `LLM_SERVICE_ERROR` | 503 | 错误提示："AI服务暂时不可用，请稍后重试" |
| `NETWORK_ERROR` | 0 | 错误提示："网络连接异常，请检查网络设置" |
| `UNAUTHORIZED` | 401 | 错误提示："登录已过期，请重新登录" |
| `FORBIDDEN` | 403 | 错误提示："无权执行此操作" |
| `INTERNAL_ERROR` | 500 | 错误提示："系统繁忙，请稍后重试" |
| `UPSTREAM_ERROR` | 500 | 错误提示："AI 服务内部异常，请稍后重试" |
