### 1.2 降级策略

**示例 — 三级降级链**：

```
CoPawAgent.ask(query, session_id)
  ├─[1] CoPaw 桥接 → answer_source='copaw', llm_used=True
  ├─[2] 百炼 DashScope → answer_source='bailian', llm_used=True
  └─[3] 离线演示兜底 → answer_source='demo', llm_used=False
```

| 约束 | 说明 |
|------|------|
| 三级降级**不可跳级** | 必须按 CoPaw → 百炼 → Demo 顺序尝试 |
| 降级**静默执行** | 不抛异常，通过 `answer_source` 标识 |
| Demo 模式**始终可用** | 无外部依赖，纯字符串拼接 |

## 2. 可用性需求

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 服务可用性 | **99%**（教学环境） | 单实例 Flask，不设 HA |
| MTTR | < 5 分钟 | 重启 Flask 进程即可恢复 |
| 故障模式：LLM服务不可用 | 自动降级到Demo模式 | 无需人工干预 | `03` 5 风险1 |
| 故障模式：股票数据源异常 | 展示缓存数据 + 提示 | 手动刷新重试 | `03` 5 风险3 |
| 故障模式：PDF解析失败 | 支持手动重解析 | 用户触发重试 | `03` 5 风险2 |
| 故障模式：外部模型响应超时 | 切换备用模型供应商 | 自动重试，用户无感知 | `03` 5 风险1 |
| 故障模式：PDF解析效果不稳定 | 多解析方案并行+规则兜底 | 自动切换OCR/表格识别 | `03` 5 风险2 |
| 故障模式：向量检索服务异常 | 降级为关键词检索 | 保留原文定位能力 | `05` AC-004 |
| 故障模式：文件存储损坏 | 从备份恢复或重新上传 | 管理员介入修复 | `04` 3.1 |
| 故障模式：数据库连接失败 | 返回503错误，提示服务维护 | 自动重连机制 | `04` 3.1 |

## 3. 可观测性需求

### 3.1 链路追踪

| 项 | 规格 |
|----|------|
| traceId 格式 | `tr_{uuid.hex}`（32 位十六进制） |
| 注入位置 | 所有成功响应顶层 + 错误响应 `error.traceId` |
| 上游透传 | 优先复用请求头 `X-Trace-Id`，缺省时本地生成 |

### 3.2 错误码体系

| HTTP | `error.code` | 触发条件 |
|------|--------------|----------|
| 400 | `EMPTY_QUERY` | query 为空/空白/null |
| 400 | `INVALID_QUERY` | query >500 字符 或 缺少 session_id |
| 400 | `QUERY_TOO_SHORT` | query 小于 1 字符 |
| 400 | `MISSING_FIELD` | 请求体缺少必填字段 |
| 404 | `REPORT_NOT_FOUND` | 研报 report_id 不存在或已删除 |
| 404 | `SESSION_NOT_FOUND` | 会话 session_id 不存在或已删除 |
| 404 | `STOCK_NOT_FOUND` | 股票代码不存在 |
| 413 | `FILE_TOO_LARGE` | 上传文件 > 50MB |
| 415 | `INVALID_FILE_TYPE` | 上传文件格式非 PDF/HTML |
| 422 | `UPLOAD_FAILED` | 文件写入磁盘失败 |
| 422 | `PARSE_FAILED` | PDF/HTML 解析失败 |
| 422 | `COMPARE_LIMIT_EXCEEDED` | 对比研报数量不在 2–5 范围 |
| 422 | `COMPARE_SAME_COMPANY_ONLY` | 对比研报对应公司不一致 |
| 422 | `CROSS_COMPANY_COMPARE` | 同 `COMPARE_SAME_COMPANY_ONLY`，内部 Storage 层抛出时使用 |
| 429 | `RATE_LIMIT_EXCEEDED` | 请求频率超限 |
| 500 | `UPSTREAM_ERROR` | Agent 编排内部异常 |
| 500 | `INTERNAL_ERROR` | 其他未预期服务端错误 |
| 503 | `LLM_SERVICE_ERROR` | 所有 LLM 提供商均不可用，降级失败 |
| 503 | `STOCK_DATA_ERROR` | 股票数据源异常 |
| 504 | `PARSE_TIMEOUT` | 研报解析超时（超过 30s） |

## 4. 安全需求

| 项 | 当前实现 | 说明 |
|----|----------|------|
| API 认证 | 无（开放访问） | Workshop 教学版 |
| CORS | 全开放 | 生产需白名单 |
| 速率限制 | 无 | 教学场景不限速 |
| 密鑰管理 | 环境变量 `.env`，禁止硬编码 | 生产需 Vault / Secrets Manager |
| 输入校验 | 基础长度校验 | 生产需完整参数校验 + 防注入 |
| 文件上传 | 格式/大小校验（PDF/HTML，≤50MB） | 生产需病毒扫描 + 内容安全检测 |
| 数据传输 | HTTP 明文（开发环境） | 生产需 HTTPS（TLS 1.3）全链路加密 |
| 日志审计 | 无 | 生产需敏感操作写入审计日志，保留 90 天 |

### 4.1 敏感数据保护

| 数据类型 | 保护方式 |
|----------|----------|
| LLM API Key | 环境变量存储（`.env`），禁止硬编码 | `04` 4 |
| 用户查询 | 仅存储在本地 JSON，不传输到第三方（LLM 除外） | `04` 4 |
| 研报文件 | 本地存储，不上云 | `04` 4 |
| 向量索引 | 本地存储 | `04` 4 |
| 用户标注数据 | 本地加密存储，不上云 | `05` AC-009 |
| 收藏数据 | 本地存储，支持导出备份 | `05` AC-008 |
| 会话历史 | 本地存储，用户可删除 | `05` AC-010 |
| 股票缓存数据 | 内存缓存30秒，不持久化 | `04` 3.5 |

## 5. 数据约束

### 5.1 输入约束

| 约束 | 规格 | 来源 |
|------|------|------|
| `query` 最小长度 | 1 字符 | `04` R-02 |
| `query` 最大长度 | 500 字符 | `04` R-02 |
| `session_id` 必填 | 非空字符串 | `04` R-01 |
| `title` 默认值 | "新会话" | `05` AC-010 |
| 会话标题生成 | 基于首条问题自动截取前20字符 | `05` AC-010-1 |
| `tags` 格式 | 字符串数组 | 收藏标签，最多5个，每个≤10字符 |

### 5.2 类型约束

| 字段 | 类型 | 约束 |
|------|------|------|
| `llm_used` | `boolean` | 必须为 True/False，不得为字符串 | `04` R-01 |
| `answer_source` | `string` | 枚举：copaw \| bailian \| demo | `04` R-01 |
| `rating` | `string` | 枚举：买入/增持/中性/减持/卖出 | `04` 3.2 |
| `status` | `string` | 枚举：pending/processing/completed/failed | `05` AC-002-4 |
| `target_price` | `number\|null` | 必须 ≥ 0 | `04` 3.2 |
| `page` | `integer` | 必须 ≥ 1 | `05` AC-002-5 |
| `color` | `string` | 枚举：yellow/green/blue/pink | `05` AC-009 |
| `file_type` | `string` | 枚举：pdf/html | `05` AC-001-2 |
| `file_size` | `number` | 必须 ≤ 50MB | `05` AC-001-2 |
| `batch_count` | `integer` | 必须 ≤ 10 | `05` AC-001-3 |
| `compare_count` | `integer` | 必须 2-5 | `05` AC-005-1 |
| `tag_length` | `integer` | 每个标签 ≤ 10字符 | `05` AC-008 |
| `tags_count` | `integer` | 最多5个标签 | `05` AC-008 |
