# 04-产品需求说明

## 1. 愿景与目标

构建**基金投资分析 Demo 应用**，帮助市场营销部门在**选品决策过程中**快速获取国际局势动态、筛选表现优异的主题基金和小规模基金，并通过可视化时间轴直观分析局势与基金表现的关联。

← 03 §1 痛点：国际局势信息分散、主题基金筛选困难、小规模优质基金发现难、局势与基金表现关联缺乏可视化

## 2. 目标用户

| 角色 | 使用目标 | 使用频率 |
|------|----------|----------|
| 市场营销部门选品人员 | 快速识别当前阶段表现优秀的基金产品，辅助选品决策 | 每周 2-3 次 |
| 基金研究员（可选） | 分析国际局势对基金表现的影响 | 每周 1-2 次 |

← 03 §2 提案概要

## 3. 核心业务场景

| 场景编号 | 场景名 | 用户路径 |
|----------|--------|----------|
| SC-001 | 查看局势动态 | 用户打开应用 → 查看美伊战争最新动态时间轴 → 了解当前局势分值 |
| SC-002 | 浏览主题基金 | 用户查看各主题（原油、黄金、煤炭、电力、天然气）表现最优基金 → 对比基金业绩 |
| SC-003 | 查看小规模基金推荐 | 用户查看精选小规模基金列表 → 了解基金规模、业绩表现 |
| SC-004 | 分析局势与基金关联 | 用户在时间轴上查看每周局势分值与基金表现的关联趋势 |

→ 05 §2 用户故事

## 4. 产品规则

| 规则编号 | 规则内容 | 来源 |
|----------|----------|------|
| R-01 | 应用以**周**为单位展示时间轴，每周为一个节点 | 原型图 |
| R-02 | 每个节点展示：**局势分值**（0-100，带颜色标识）、**局势动态摘要** | 原型图 |
| R-03 | **主题基金区域**：展示原油、黄金、煤炭、电力、天然气各主题表现最优基金，含基金名称、涨跌幅 | 原型图 |
| R-04 | **精选小规模基金区域**：展示从广发、易方达、富国、南方、华夏筛选的规模 5 亿以下、业绩优异基金，含基金名称、涨跌幅 | 原型图 |
| R-05 | 小规模基金不能与主题基金重叠 | 03 §3 范围边界 |
| R-06 | 支持按周切换查看历史数据 | 原型图 |
| R-07 | 局势分值以 Polymarket 停战概率为参考 | 03 §2 提案概要 |
| R-08 | 主题基金筛选可先根据基金名称确定 | 补充需求 |

← 03 §4 成功标准、§3 范围边界
